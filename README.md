# 灵芽 Lingya

灵芽是一个偏收藏与沉淀的灵感工具。

这版是一个可运行的 `Next.js + PWA` 原型，已经实现了：

- 首页灵感卡片流
- 随机卡片配色系统
- 新增灵感流程
- AI 分析的前端 mock 流程
- 详情页与统一笔记弹窗
- 月度总结页
- 我的页
- 本地存储
- 基础 PWA 配置

## 本地开发

```bash
npm install
npm run dev
```

打开 `http://localhost:3000`

## 当前技术栈

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- Lucide React
